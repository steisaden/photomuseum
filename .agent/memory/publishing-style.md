# Publishing Style Guide

Default formats:
- TikTok/IG Reel: 30–60s, hook → proof → steps → CTA.
- Carousel: 7–10 slides, 1 idea per slide, strong headings.
- X thread: 6–12 posts, numbered, crisp.
- LinkedIn: story + takeaway + proof + question.

Rules:
- No overpromising.
- Show receipts (screenshots, code, before/after).
- One CTA per post.
- No secrets (redact keys/tokens).
